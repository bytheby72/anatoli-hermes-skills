- "Which skills have no tests?"
- "Which scripts read configuration files?"

- "Which scripts can send data externally?"
- "Which files have network access and file read?"

- "Which cron jobs run with high privileges?"

## Operational Questions

- "Which cron jobs point to missing files?"
- "Which skills are missing documentation?"
- "Which processes consume the most CPU?"
- "What cleanup candidates exist?"
- "Which systemd services are failed?"

## Server Questions

- "What is loading the server?"
- "Which directories consume the most disk?"
- "Which log files are largest?"
- "Which Docker containers are stopped?"

## Maintenance Questions

- "What changed in the last 7 days?"
- "Which skills have not been modified recently?"
- "Which scripts have no error handling?"
- "Which files are the largest in the workspace?"
